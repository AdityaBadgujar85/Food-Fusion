# 🍽️ Food-Fusion — Food Ordering E-Commerce Website

Food-Fusion is a **Real-Life Style Food Ordering Web Application** built using **React**, **React Router**, **Redux**, **Axios**, and **Lazy Loading**.  
It allows users to **browse food items, view details, and add items to cart**, similar to Swiggy / Zomato UI flow.

## 📌 Features

### ✅ **Browse Delicious Foods**
- Displays food menu fetched using **Axios** from a real food API.
- Food cards show **Image, Name, Short Description, Price**.

### 🛒 **Shopping Cart System (Redux)**
- Add to cart from product list or detail page.
- Increase / Decrease quantity.
- Remove item from cart.
- Real-time **total bill calculation**.

### 🔄 **React Routing**
- Home Page  
- Menu / Products Page  
- Cart Page  
- About Page  
- Contact Page  

### 🧱 **Redux State Management**
- Cart state is updated globally across all components.

### 💤 **Lazy Loading**
- Pages are loaded lazily for **performance optimization**.

### 🌐 **Axios API Integration**
- Fetches live food list from **public food API** *(e.g., themealdb.com)*.

### 🎨 **Responsive UI**
- Works smoothly on **Desktop, Tablet & Mobile**.

### 🎯 **Professional Layout**
- Includes **Header & Footer** with Logo + Navigation Links.
